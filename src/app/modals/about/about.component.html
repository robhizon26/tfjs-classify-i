This is a mobile app made from Ionic that can test out TFJS model files without modifying anything on the code. Just copy the model files on a particular path, run and build it, create the Android (or iOS) project, and run it on Android studio (or XCode) and you will be able to test it out on your mobile device.
Ionic is a mobile development technology that can maintain one codebase for a project that is applicable on both Android and iOS. TFJS (Tensorflow for Javascript) is very much compatible with this technology because it is made for Javascript coding.
The app name 'TFJS Classify I' means TFJS-Tensorflow JS(Javascript); Classify-Image Classifiction, which is a type of deep learning application on AI; I-Ionic.
The camera view used on this app is not directly from native camera of the device but from the video that is fed to the HTMLVideoElement from the native camera.
If you want to know more about the details of this app, such as to what path should the model files be copied, you could go here.
If you want to generate TFJS web-format model files, I made a simple script on Google Colab to make these files here and the blog here . It will generate these model files (binary weight and model.json files) plus a labels.txt file on your Google Drive.
The app already contains several of the model files I created found in that particular path. You could also see the list on the Model modal screen in the app.
